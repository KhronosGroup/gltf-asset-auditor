Welcome to the Khronos 3D Commerce Asset Validator!

This is a web-based example implementation of the 3DC-validator. It allows you to quickly check a glTF file for a specific use case, defined by a schema. Use cases could include publishing to a specific retailer or advertising platform. The Khronos Group invites companies to provide a link to their schemas for us to publish.

You can use the same npm package in your own Node or browser-based project for automated 3D asset validation. See https://github.com/KhronosGroup/3DC-Validator/tree/main/cli-example.
